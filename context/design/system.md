# Design System

## Aesthetic Identity
Applicant is a premium, developer-focused "Dev-Tool", prioritizing density and clarity over flashy animations. It feels professional, fast, and un-opinionated.

## Typography
- **Core Font:** Geist (by Vercel). Unapologetically modern and clean.

## Iconography
- **Library:** Lucide React.
- **Rule:** All icons must be unified at `stroke-width: 1.2px` for a thin, sharp look.

## Color Palette & Theme
- **Theme:** Default Light Mode (MVP). Very subtle grays and high-contrast blacks. Monochromatic primary actions (black/white buttons).
- UI Elements are governed by **Shadcn** defaults (Zinc/Slate base colors).

## UI Patterns
- **High-Density Tables:** Minimal padding, clear borders.
- **Split Editors:** Code-editor standard themes (e.g., Monaco/VS Code lite visually).
