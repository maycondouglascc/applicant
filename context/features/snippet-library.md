# Snippet Library

## Core Specification
The Library is where the user manages their "Master Profile" as a collection of Markdown snippets.

## Features
- **Snippet Management:** CRUD operations for individual markdown files (e.g., "Main CV Base", "React Case Study", "Leadership Experience").
- **Persona Settings:** Global setting to switch the AI Voice (e.g., `Professional/Humble` vs `Aggressive/Startup`).
- **Tagging (Optional Future):** Ability to tag snippets to help the AI selector (e.g., `#frontend`, `#management`).
